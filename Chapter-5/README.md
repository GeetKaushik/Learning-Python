# Chapter - 5 : Dictionary and Sets

## In this chapter we will learn about Dictionary and Sets

- ### Dictinoary
  - `Dictionary` is a collection of key-value pairs
    - Example: 
      ```py
        marks = {
          "Geet" : 100,
          "Sanu" : 90,
          "Mukul" : 80,
          "Ishan" : 70
          }
      ```
  - `Dictionary` cannot contain duplicate Keys.
  - `Dictionary` are mutable, unodered, indexed.

- ### Sets
  - `Sets` is an unordered collection of unique elements.
    - Example : 
    ```py
    fruits = {"apple", "banana", "cherry"}
    ```
  - `Sets` are mutable, unodered, unindexed
